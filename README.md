# Homebrew Cue

Homebrew tap for [Cue CLI](https://getcue.net/cli) — manage events from your terminal.

## Install

```
brew install vegardlu/cue/cue-cli
```

## Update

```
brew upgrade cue-cli
```

## What is Cue?

[Cue](https://getcue.net) is an event coordination platform. Create events, propose times, invite friends, and let everyone vote on what works. Available on [iOS](https://apps.apple.com/app/cue-event-planner/id6741188498), [web](https://getcue.net), and now the command line.

## Quick Start

```bash
cue auth login          # Sign in with Google or email
cue events list         # See your events
cue events create       # Create a new event
cue respond 42          # Respond to an invitation
cue watch               # Stream live updates
```
